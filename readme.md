```
| task1

| - test.ts
| task2

| task3

| - test.ts
| index.ts // << - Entry point
```

## Getting started

Install all the library

```
$ yarn
```

Run test

```
$ yarn test
```

Start the app, see the result

```
$ yarn start
```

About how it works
Inside each task/index.ts, the implementation, will have the comment show how it works
